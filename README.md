MY VIM CONFIGURATION
====================

These are my configuration files for VIM and how I use it.

Setup
-----

I symlinked .vim/.vimrc and .vim/.gvimrc to the same files in the ~/ directory:

- ln -nfs ~/.vim/.vimrc ~/.vimrc
- ln -nfs ~/.vim/.gvimrc ~/.gvimrc

Contains
--------

- Haml/Sass support
